# 一、数组

| **类型**     | **描述**                       |
| ------------ | ------------------------------ |
| 数组 `Array` | 数组可以在一个变量中存储多个值 |

## 1、创建空数组

```php
$arr = array();
var_dump( $arr );
$arrs = [];
var_dump( $arrs );
```

```
array(0) { } array(0) { }
```



## 2、创建索引数组

- 数组之间用,隔开

```php
$arr = array(
    '无意白纸',
    '男',
    '18'
);
var_dump( $arr );
```

```
array(3) { [0]=> string(12) "无意白纸" [1]=> string(3) "男" [2]=> string(2) "18" }
```



## 3、创建关联数组

```php
$arr = [
	'Name' => '无意白纸',
	'Sex' => '男',
	'Old' => '18'
];
var_dump( $arr );
```

```
array(3) { ["Name"]=> string(12) "无意白纸" ["Sex"]=> string(3) "男" ["Old"]=> string(2) "18" }
```



## 4、输出数组值

```php
$arr = [
        '无意白纸',
        '男',
        '18'
];
echo $arr[0];
echo '<hr>';
echo $arr[1];
echo '<hr>';
echo $arr[2];
echo '<hr>';


$arrs = [
        'Name' => '无意白纸',
        'Sex' => '男',
        'Old' => '18'
];
echo $arrs['Name'];
echo '<hr>';
echo $arrs['Sex'];
echo '<hr>';
echo $arrs['Old'];
echo '<hr>';
```

```
无意白纸
男
18
无意白纸
男
18
```



## 5、打印数组 `print_r`

```php
    $arr = [
        '无意白纸',
        '男',
        '18'
    ];
    print_r($arr);
```

```
Array ( [0] => 无意白纸 [1] => 男 [2] => 18 )
```



## 6、`php` 连接符

```php
$var1 = 'PHP讲师';
$var2 = '欧阳';
var_dump( $var1 . $var2 );
var_dump( $var1 . '：' . $var2 );
```

## 二、`php` 多维数组

### 1、二维数组

```php
$arr = array(
    array(
        'name' => '欧阳',
        'school'  => 'PHP中文网'
    ),
    array(
        'name' => '西门',
        'school'  => 'PHP中文网'
    ),
    array(
        'name' => '灭绝',
        'school'  => 'PHP中文网'
    )
)
var_dump($arr);
print_r($arr);
```

### 2、三维数组

```php
$arr = [
    [
        'name' => '欧阳',
        'school'  => 'PHP中文网',
        'gongfu' => [
            'PHP',
            '小程序',
            'layui',
            'Thinkphp'
        ]
    ],
    [
        'name' => '西门',
        'school'  => 'PHP中文网',
        'gongfu' => [
            'PHP',
            'Thinkphp',
            'Laravel',
            '实战项目'
        ]
    ],
    [
        'name' => '灭绝',
        'school'  => 'PHP中文网',
        'gongfu' => [
            'HTML',
            'PHP',
            'layui',
            'Thinkphp'
        ]
    ]
];
var_dump($arr);
print_r($arr);
```

> 备：数组最好不要超过3层

### 3、多维数组访问

```php

        'gongfu' => [
            'PHP',
            '小程序',
            'layui',
            'Thinkphp'
        ]
    ],
    [
        'name' => '西门',
        'school'  => 'PHP中文网',
        'gongfu' => [
            'PHP',

            'Laravel',
            '实战项目'
        ]
    ],
    [
        'name' => '灭绝',
```

## 三、`php` 数组循环

### 1、`foreach`

```php
$arr = array(
    'ouyang' => '欧阳',
    'ximen' => '西门',
    'miejue' => '灭绝'
);
foreach( $arr as $v ){
    echo $v;
    echo '<hr>';
}
```

### 2、`key` 和 `value`

> $k 和 $v，变量名，可以自定义

```php
示例1：
$arr = array[
    'ouyang' => '欧阳',
    'ximen' => '西门',
    'miejue' => '灭绝'
];
foreach( $arr as $k=>$v ){
    echo $k . ' --- ' . $v;
    echo '<hr>';
}
示例2：
$arr = [
    '欧阳',
    '西门',
    '灭绝'
];
foreach( $arr as $k=>$v ){
    echo $k . ' --- ' . $v;
    echo '<hr>';
}
```
