# 一、数组

| **类型**     | **描述**                       |
| ------------ | ------------------------------ |
| 数组 `Array` | 数组可以在一个变量中存储多个值 |

## 1、创建空数组

```php
    $arr = array();
    var_dump( $arr );
    $arrs = [];
    var_dump( $arrs );
```

```
array(0) { }
array(0) { }
```



## 2、创建索引数组

```php
    $arr = array(
        '无意白纸',
        '男',
        '18'
    );
    var_dump( $arr );
```

```
array(3) { [0]=> string(12) "无意白纸" [1]=> string(3) "男" [2]=> string(2) "18" }
```

### PS : 每一个数组里面的值都有一个默认的键从0开始

### 键也可以自定义用 => 号链接

## 3、创建关联数组

```php
    // 创建关联数组
    $arr = array(
            'Name' => '无意白纸',
            'Sex' => '男',
            'Year' => '18'
    );
    var_dump($arr);
```

```
array(3) { ["Name"]=> string(12) "无意白纸" ["Sex"]=> string(3) "男" ["Year"]=> string(2) "18" }
```



## 4、输出数组值

```php
    //输出数组值
    $arr = [
        '无意白纸',
        '男',
        '18'
    ];
    echo $arr[0];
    echo '<hr>';
    echo $arr[1];
    echo '<hr>';
    echo $arr[2];
    echo '<hr>';

    $arr_s = [
            'Name' => '无意白纸',
            'Sex' => '男',
            'Year' => 18
    ];

    echo '我的名字叫: '.$arr_s['Name'];
    echo '<hr>';
    echo '我的性别是: '.$arr_s['Sex'];
    echo '<hr>';
    echo '我今年: '.$arr_s['Year'].'岁了';
```

```
无意白纸
男
18
我的名字叫: 无意白纸
我的性别是: 男
我今年: 18岁了
```



## 5、打印数组 `print_r`

- 和 var_dump 不同的是 prrint_r 不会打印数组中值的类型

```php
    // 打印数组 print_r
    $arr = [
        '无意白纸',
        '男',
        '18'
    ];
    print_r($arr);
```

```
Array ( [0] => 无意白纸 [1] => 男 [2] => 18 )
```



## 6、`php` 连接符

```php
    $var1 = 'Name';
    $var2 = '无意白纸';
    var_dump( $var1 . $var2 );
    var_dump( $var1 . '：' . $var2 );
```

### 3、多维数组访问

```php
$arr = [
    [
        'name' => '欧阳',

            'PHP',
            '小程序',
            'layui',

        'school'  => 'PHP中文网',
        'gongfu' => [
            'PHP',
            'Thinkphp',
            'Laravel',
            '实战项目'
        ]
    ],
    [
        'name' => '灭绝',
        'school'  => 'PHP中文网',
        'gongfu' => [
            'HTML',
            'PHP',
            'layui',
            'Thinkphp'
        ]
    ]
]

```

> 备：数组访问时，层次不要弄错

## 三、`php` 数组循环

### 1、`foreach`

```php
$arr = array(
    'ouyang' => '欧阳',
    'ximen' => '西门',
    'miejue' => '灭绝'
);
foreach( $arr as $v ){
    echo $v;
    echo '<hr>';
}
```

### 2、`key` 和 `value`

> $k 和 $v，变量名，可以自定义

```php
示例1：
$arr = array[
    'ouyang' => '欧阳',
    'ximen' => '西门',
    'miejue' => '灭绝'
];
foreach( $arr as $k=>$v ){
    echo $k . ' --- ' . $v;
    echo '<hr>';
}
示例2：
$arr = [
    '欧阳',
    '西门',
    '灭绝'
];
foreach( $arr as $k=>$v ){
    echo $k . ' --- ' . $v;
    echo '<hr>';
}
```
